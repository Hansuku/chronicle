# Earth texture sources

These files are local, presentation-optimized copies used by the Three.js globe.

- `earth_day_8192.jpg` — NASA Goddard Space Flight Center, *Blue Marble: Land Surface, Ocean Color and Sea Ice*, by Reto Stöckli and the MODIS team. The 8192×4096 NASA image is mirrored by Wikimedia Commons as `Nasa_land_ocean_ice_8192.jpg`. Original NASA asset: <https://eoimages.gsfc.nasa.gov/images/imagerecords/57000/57730/land_ocean_ice_8192.png>. NASA source page: <https://science.nasa.gov/resource/blue-marble/>.
- `earth_night_8192.jpg` — NASA GSFC / NOAA NGDC / DMSP, *Earth's City Lights 1994*. Data courtesy Marc Imhoff (NASA GSFC) and Christopher Elvidge (NOAA NGDC); image by Craig Mayhew and Robert Simmon (NASA GSFC). Local file is the 8192×4096 Wikimedia mirror. NASA SVS source and credits: <https://svs.gsfc.nasa.gov/30003>.
- `earth_topography_5400.jpg` — NASA Earth Observatory, *Blue Marble: Next Generation Topography*, 5400×2700. Imagery by Jesse Allen using NASA radar-derived elevation data. Source: <https://science.nasa.gov/earth/earth-observatory/blue-marble-next-generation/topography-bathymetry-maps/>.
- `earth_clouds_2048.jpg` — NASA Goddard Space Flight Center, *Blue Marble: Clouds*, by Reto Stöckli and the MODIS team. Source: <https://visibleearth.nasa.gov/images/57747/blue-marble-clouds>.
- `earth_specular_2048.jpg` — Three.js example Earth specular map. Source: <https://threejs.org/examples/textures/planets/earth_specular_2048.jpg>.

NASA imagery is used according to the NASA media usage guidelines; NASA endorsement is not implied. Wikimedia is only used as a faster mirror for the two credited NASA/NOAA image files above.
